🔍 Guide: Trouver ta Web API Key Firebase

Étape 1: Ouvre Firebase Console

Va à cette URL:

https://console.firebase.google.com/project/hivoraa-323d9/settings/general

Étape 2: Va à "Project Settings"

En haut à droite, clique sur l'icône ⚙️ (paramètres) → Project Settings

Étape 3: Scroll jusqu'à "Your apps"

Tu verras une section "Your apps" avec:

hivoraa-323d9 Web SDK setup and configuration

Étape 4: Clique sur "Web"

Ou cherche le bloc qui commence par:

const firebaseConfig = { apiKey: "AIzaSy...", ← C'EST CETTE CLÉ! authDomain: "hivoraa-323d9.firebaseapp.com", ... }

Étape 5: Copie ta clé complète

La clé ressemblera à:

AIzaSyD_XXXXXXXXXXXXXXXXXXXXXX_XXXXXXXXXXXX

Elle commence toujours par AIzaSy

✅ Maintenant, mets-la dans debug.html

Édite /public/debug.html ligne ~27:

const firebaseConfig = { apiKey: "AIzaSyD_XXXXXXXXXXXXXXXXXXXXXX_XXXXXXXXXXXX", // ← Remplace authDomain: "hivoraa-323d9.firebaseapp.com", projectId: "hivoraa-323d9", };

⚠️ Important

  • Jamais utilise ta clé Admin SDK (celle dans .env) côté client
  • La Web API Key est publique, c'est normal
  • Après avoir changé la clé, recharge http://localhost:5000/debug.html
